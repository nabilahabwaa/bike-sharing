# Sharing Data Analysis Project 

## Deskripsi
Proyek ini bertujuan untuk menganalisis tren penyewaan sepeda berdasarkan profil pengguna (Casual vs Registered) dan pengaruh kondisi cuaca pada hari kerja. Analisis ini divisualisasikan melalui dashboard interaktif menggunakan Streamlit.

## Setup Environment - Shell/Terminal
```
mkdir proyek_analisis_data
cd proyek_analisis_data
pipenv install
pipenv shell
pip install -r requirements.txt
```
## Run streamlit app
```
streamlit run dashboard/dashboard.py
```
